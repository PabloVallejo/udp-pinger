### Usage manual

#### Quick start

To get started, [download](https://github.com/PabloVallejo/udp-pinger/archive/master.zip) this repository and run the server code like this:

```bash
# Cd in to repo.
$ cd udp-pinger

# Start the UDP server.
$ python server.py
```

Then, in a new terminal tab run the client code.

```bash

# Run client.
$ python client.py
```
